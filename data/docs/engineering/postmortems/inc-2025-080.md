# Postmortem INC-2025-080: config-service — tenant config fetch errors

**Date:** 2025-07-03 · **Service:** `config-service` · **Severity:** SEV2 · **Doc ID:** INC-2025-080

## Summary
On 2025-07-03, `config-service` experienced tenant config fetch errors.

## Root cause
The root cause was a cache stampede after a deploy.

## Resolution
The team added request coalescing and a warm cache. Mitigation followed the incident runbook (ENG-INC-009).

## Action items
- Add or improve monitoring to catch this earlier.
- Document the failure mode in the `config-service` runbook.
- Review related services for the same risk.

This was a blameless postmortem reviewed with the owning team.
