# Postmortem INC-2025-064: metrics-store — metric write failures for one shard

**Date:** 2025-05-29 · **Service:** `metrics-store` · **Severity:** SEV2 · **Doc ID:** INC-2025-064

## Summary
On 2025-05-29, `metrics-store` experienced metric write failures for one shard.

## Root cause
The root cause was a full disk on a ClickHouse node.

## Resolution
The team expanded storage and added a disk-usage alert. Mitigation followed the incident runbook (ENG-INC-009).

## Action items
- Add or improve monitoring to catch this earlier.
- Document the failure mode in the `metrics-store` runbook.
- Review related services for the same risk.

This was a blameless postmortem reviewed with the owning team.
