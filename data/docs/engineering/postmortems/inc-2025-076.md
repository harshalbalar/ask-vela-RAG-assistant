# Postmortem INC-2025-076: trace-collector — 10% of spans dropped

**Date:** 2025-06-25 · **Service:** `trace-collector` · **Severity:** SEV2 · **Doc ID:** INC-2025-076

## Summary
On 2025-06-25, `trace-collector` experienced 10% of spans dropped.

## Root cause
The root cause was backpressure from a slow downstream writer.

## Resolution
The team scaled the writer and added buffering. Mitigation followed the incident runbook (ENG-INC-009).

## Action items
- Add or improve monitoring to catch this earlier.
- Document the failure mode in the `trace-collector` runbook.
- Review related services for the same risk.

This was a blameless postmortem reviewed with the owning team.
