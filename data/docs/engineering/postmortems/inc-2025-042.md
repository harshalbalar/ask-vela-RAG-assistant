# Postmortem INC-2025-042: billing-service — duplicate invoices sent to 14 customers

**Date:** 2025-04-02 · **Service:** `billing-service` · **Severity:** SEV2 · **Doc ID:** INC-2025-042

## Summary
On 2025-04-02, `billing-service` experienced duplicate invoices sent to 14 customers.

## Root cause
The root cause was a retry without idempotency keys.

## Resolution
The team added idempotency keys to Stripe calls. Mitigation followed the incident runbook (ENG-INC-009).

## Action items
- Add or improve monitoring to catch this earlier.
- Document the failure mode in the `billing-service` runbook.
- Review related services for the same risk.

This was a blameless postmortem reviewed with the owning team.
