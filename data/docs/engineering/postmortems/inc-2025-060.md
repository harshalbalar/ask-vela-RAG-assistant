# Postmortem INC-2025-060: search-service — stale search results for 40 minutes

**Date:** 2025-05-18 · **Service:** `search-service` · **Severity:** SEV2 · **Doc ID:** INC-2025-060

## Summary
On 2025-05-18, `search-service` experienced stale search results for 40 minutes.

## Root cause
The root cause was an index rebuild job that silently failed.

## Resolution
The team added job alerting and a freshness check. Mitigation followed the incident runbook (ENG-INC-009).

## Action items
- Add or improve monitoring to catch this earlier.
- Document the failure mode in the `search-service` runbook.
- Review related services for the same risk.

This was a blameless postmortem reviewed with the owning team.
