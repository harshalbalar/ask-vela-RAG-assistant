# Postmortem INC-2025-031: query-service — elevated query timeouts for EU tenants

**Date:** 2025-03-04 · **Service:** `query-service` · **Severity:** SEV2 · **Doc ID:** INC-2025-031

## Summary
On 2025-03-04, `query-service` experienced elevated query timeouts for EU tenants.

## Root cause
The root cause was a slow ClickHouse query plan after a schema migration.

## Resolution
The team added a missing index and a query timeout guard. Mitigation followed the incident runbook (ENG-INC-009).

## Action items
- Add or improve monitoring to catch this earlier.
- Document the failure mode in the `query-service` runbook.
- Review related services for the same risk.

This was a blameless postmortem reviewed with the owning team.
