# Postmortem INC-2025-072: export-service — export jobs stuck in pending

**Date:** 2025-06-17 · **Service:** `export-service` · **Severity:** SEV2 · **Doc ID:** INC-2025-072

## Summary
On 2025-06-17, `export-service` experienced export jobs stuck in pending.

## Root cause
The root cause was a deadlock in the job scheduler.

## Resolution
The team fixed the lock ordering and added a watchdog. Mitigation followed the incident runbook (ENG-INC-009).

## Action items
- Add or improve monitoring to catch this earlier.
- Document the failure mode in the `export-service` runbook.
- Review related services for the same risk.

This was a blameless postmortem reviewed with the owning team.
