# Password & MFA Policy

**Owner:** Security · **Last updated:** 2025-01-25 · **Doc ID:** IT-PWD-017

## Passwords
Passwords must be at least **14 characters**. Vela Cloud requires the company password manager
(**1Password**) for all work credentials. Reusing personal passwords for work accounts is prohibited.

## Multi-factor authentication
**MFA is mandatory** on all accounts via Okta Verify or a hardware key (YubiKey). SMS-based MFA is
not permitted. New laptops are enrolled in MFA during provisioning.

## Rotation
We do **not** force periodic password rotation. Passwords are only reset on suspected compromise.

## Lost device
Report a lost or stolen device immediately to `#it-help` so credentials and sessions can be revoked.
