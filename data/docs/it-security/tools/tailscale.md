# Tool Guide: Tailscale VPN

**Owner:** IT · **Doc ID:** IT-TOOL-TAILSCALE

## What it is
Tailscale VPN is used for secure access to internal systems.

## Getting access
To get access: install from the IT self-service portal and sign in with Okta.

## Common issue
Note: VPN-110 means Tailscale is not running. For anything else, message `#it-help` in Slack.
